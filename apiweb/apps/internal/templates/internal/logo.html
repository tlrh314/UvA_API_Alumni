API logo

Since November 2011, the API has its own "official" logo. Designed by Christian Thalmann with help from several interested colleagues, it can (and should!) be used on letters, posters or in email.

Currently we have a template keynote letterhead for MAC and a template for LaTeX. Both contain the UvA and API logo. People are of course free to implement their own and if wanted, put them on this website. Let the current webpage maintainer(s) know if you like to do so.

How to use the logo:

  • The examples below are PDF and PNG files. The PDF version should be used whenever possible, since it contains smooth vector graphics rather than pixels. This is particularly important for printed matter.
  • The logo comes in two different color schemes: black and white. The black scheme should be used by default. The black version is intended for black-and-white printing, whereas the white version can be placed on a dark background (on a slide or poster).
  • The logo comes in three different arrangements: default, big, and wide. The default arrangement should be used by default. The other two versions are available for special formating needs (e.g., the "big" version can be used to emphasize the graphics over the text, and the "wide" version may fit in well with other wide-format logos on a poster).
  • The logo mark (the dome graphic) may be used without the text part if necessary, since it already includes the "API" letters. However, please do not add "Anton Pannekoek Institute for Astronomy" in another font.

Some technical details:

  • The font used is the small-caps form of EB Garamond (free).
  • The colors are black, white and crimson, with RGB codes 000000, ffffff and fe0012.
  • If you need something like a JPEG, EPS, or TIFF, use a tool like PhotoShop or Gimp to convert the file. In the future, more formats may be put here as well.
