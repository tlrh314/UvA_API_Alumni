This form allows you to register for the upcoming stargazing event. You can select various activities, and create your own program. This program is valid for the number of people registered at the same time. You will receive an e-mail as soon as you register. Please bring a copy of this e-mail to the stargazing event.

We keep you informed with the latest developments concernning the API stargazing nights via a newsletter. You can indicate in this form whether you want to receive this newsletter by email.

The activities you can choose from:

  • Lecture - Lecture about an astronomy topic
  • Quiz - Quiz about the topic of the lecture
  • Tour A/B - Tour through the domes and telescopes of the Anton Pannekoek Institute when it is cloudy
  • Stargazing A/B - Stargazing with the telescopes in the domes when it is a clear night
  • Astrocaf√© - Ask an astronomer your pressing questions when it is cloudy
  • Stargazing P - Basic stargazing with binoculars and/or small telescopes when it is a clear night
  • Stellarium - Explanations about the current night sky and observing tips
